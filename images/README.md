# Photos

Place your photos here with these exact filenames:

| File | Used in |
|------|---------|
| `photo1.jpg` | Gallery — slot 1 |
| `photo2.jpg` | Gallery — slot 2 |
| `photo3.jpg` | Gallery — slot 3 |

Any common image format works (JPG, PNG, WEBP). The gallery displays them at a 3:4 portrait ratio, so portrait-oriented photos look best.
